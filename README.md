# Drone-simulator-assignment
Game Controls / How to Play
Drone Movement
W / A / S / D – Move the drone
Q – Move the drone downward
E – Move the drone upward
Left Arrow / Right Arrow – Rotate the drone
Camera Control
Mouse Movement – Pan / look around the drone
Targeting & Combat
F – Lock onto a target
R – Release the locked target
Spacebar – Launch missile

Objective
Control the drone, locate targets, lock onto them, and launch missiles accurately.
